finderjs

Finder-like UI for viewing hierarchical content in columns

View Source Download Zip

Introduction

FinderJS allows you to browse hierarchical data in columns, similar to OS X's Finder.

Features

  • Data source: Array or Function (useful for dynamic/asynchronous content such as a remote web service).
  • Rendering: define how items are rendered.
  • Stlying: Use any CSS/framework of choice, or feel free to use the example CSS (which leverages Flexbox). Additionally, classnames are customizable.
  • Keyboard navigation: use arrows keys to navigate.
  • Events: several events are emitted to allow for custom behavior.
  • Module formats: Provided as an npm module (CommonJS),exported as the global finderjs, and as a jquery plugin.
  • File size: 4KB minified and gzipped.
  • Test coverage: 100%.
